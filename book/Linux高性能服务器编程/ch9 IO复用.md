# 第9章 I/O复用

1. I/O复用使得程序能同时监听多个文件描述符，这对提高程序的性能至关重要。通常，网络程序在下列情况下需要使用I/O复用技术：
    * 客户端程序要同时处理多个 socket
    * 客户端程序要同时处理用户输入和网络连接
    * TCP 服务器要同时处理监听 socket 和连接 socket
    * 服务器要同时处理 TCP 请求和 UDP 请求
    * 服务器要同时监听多个端口，或者处理多种服务
